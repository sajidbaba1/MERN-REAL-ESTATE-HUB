# MERN REAL ESTATE HUB

Welcome to the **MERN REAL ESTATE HUB**, a comprehensive real estate platform built with the MERN stack (MongoDB, Express, React, and Node.js). This project features real-time chat, notifications, AI assistant integration, and map-driven location searches.

## Features
- **Modern UI**: Built with React and styled with Tailwind CSS.
- **Real-time Communication**: Integrated chat and notifications using Node.js backend.
- **AI Assistant**: Powered by Gemini / OpenAI for intelligent interactions.
- **Property Maps**: Interactive maps using Google Maps API.
- **Backend Management**: Robust backend handling data, uploads, and security.

## Project Structure
- `backend-nodejs/`: Node.js backend server handling AI, chats, and notifications.
- `frontend/`: React frontend application.

## Quick Start
1. **Clone the repository.**
2. **Setup Backend**:
   - `cd backend-nodejs`
   - `npm install`
   - Configure `.env` with your credentials.
   - `npm run dev`
3. **Setup Frontend**:
   - `cd frontend`
   - `npm install`
   - Configure `.env.local`.
   - `npm run dev`

For detailed setup instructions, please refer to the `SETUP_GUIDE.md` file.
